# Gutenberg Pincode Address Block

A custom WordPress Gutenberg block that automatically fetches and populates address details based on Indian pincode input.

This block improves user experience by reducing manual data entry and ensuring accurate location details using API integration.

---

## 🚀 Features

- 📍 Auto-fetch address details using pincode
- 🧠 Intelligent autofill (State, District, City)
- ⚡ Built with Gutenberg (React-based block editor)
- 🔌 Easy API integration (India Post or custom API)
- 🎯 Improves form UX and data accuracy
- 🧩 Reusable block for any WordPress project

---

## 🛠️ Tech Stack

- WordPress Gutenberg (Block Editor)
- React.js
- JavaScript (ES6+)
- REST API / External API Integration
- PHP (for block registration if applicable)
